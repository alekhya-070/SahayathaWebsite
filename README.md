# SahayathaWebsite
# 📌 Club Website Project (MERN Stack)

Welcome to the **Club Website** project! This repository contains the codebase for our technical club's website, built using the **MERN stack** (MongoDB, Express.js, React, Node.js).

## 🛠️ Project Structure

```
club-website/
├── backend/         # Node.js + Express API
├── frontend/        # React (User Interface)
└── README.md        # Documentation
```

## 🎯 Project Goals

1. Create a fully functional **club website** showcasing our activities and updates.
2. Implement core features like **user authentication**, **contact forms**, and **dynamic content**.
3. Learn and collaborate using the **MERN** stack and **GitHub** workflows.

## 📚 Tech Stack

- **Frontend**: React + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (using Mongoose)
- **Deployment**: Vercel (Frontend), Render (Backend)

## 🔨 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-org/club-website.git
cd club-website
```

### 2. Set Up Frontend

```bash
cd frontend
npm install
npm start
```

### 3. Set Up Backend

```bash
cd backend
npm install
npm run dev
```

## 🚀 Contribution Guide

### Branch Strategy

- **main**: Stable code (production-ready)
- **dev**: Active development
- **feature/**: New features (e.g., `feature/navbar`)

### Workflow

1. Create a new branch for your task:
   ```bash
   git checkout -b feature/task-name
   ```
2. Commit changes with meaningful messages:
   ```bash
   git add .
   git commit -m "Add navbar component"
   ```
3. Push your branch and create a Pull Request:
   ```bash
   git push origin feature/task-name
   ```
